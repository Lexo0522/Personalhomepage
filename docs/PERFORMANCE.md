# 性能优化配置

## 资源优化

### 图片优化
- 使用WebP格式（支持时）
- 提供多种分辨率（响应式图片）
- 实施懒加载
- 压缩图片文件

### CSS优化
- 压缩CSS文件
- 移除未使用的CSS
- 使用CSS变量减少重复
- 关键CSS内联

### JavaScript优化
- 压缩JavaScript文件
- 代码分割和懒加载
- 使用Tree Shaking
- 移除未使用的代码

## 缓存策略

### 浏览器缓存
- 静态资源：长期缓存（1年）
- HTML文件：短期缓存（1小时）
- API响应：中等缓存（1天）

### Service Worker缓存
- 预缓存关键资源
- 动态缓存API响应
- 智能更新策略
- 离线支持

## 加载优化

### 关键渲染路径
- 优先加载首屏内容
- 延迟加载非关键资源
- 使用资源提示（preload, prefetch）
- DNS预解析

### 网络优化
- HTTP/2支持
- 资源压缩（Gzip, Brotli）
- CDN分发
- 减少HTTP请求

## 监控指标

### Core Web Vitals
- LCP（最大内容绘制）< 2.5s
- FID（首次输入延迟）< 100ms
- CLS（累积布局偏移）< 0.1

### 其他指标
- 首字节时间（TTFB）< 600ms
- 首次内容绘制（FCP）< 1.8s
- 速度指数< 3.4s
